每日 UI 挑戰 005 - APP 的標誌

APP 的標誌是品牌和產品設計的一環，需要具備那個品牌或是產品的代表性元素，跟其他的 APP 相比要能夠被人家一眼就認出來並記住，同時還要符合 IOS 以及 Android 的規範。

每日 UI 挑戰 006 - 使用者個人頁面

我以社群交友的網站和 APP 作為設計對象，在網頁的視覺上，我把重點資訊集中在螢幕的中間，其餘的圖片貼文則是橫跨整個螢幕，這樣子在搜尋特定貼文或圖片時會更加方便，基於同樣的理由，我將 APP 的圖片分為兩列。

每日 UI 挑戰 009 - 音樂播放器

在設計音樂播放器的時候，我考慮得最久的就是音量的控制軸，因為如果我設計的是 APP，手機已經有音量的控制鍵，其實不太需要再多一個控制，但是在考慮到有的使用者並不習慣按音量鍵控制音量後，我決定還是加上去。
